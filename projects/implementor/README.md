# Implementor

## Описание

Реализация утилиты для генерации Java-классов, автоматически создающих заглушки для интерфейсов и абстрактных классов.

Проект основан на интерфейсе `JarImpler` и поддерживает генерацию `.java` и `.jar` файлов.

---

## Возможности

* генерация реализации интерфейсов и абстрактных классов
* создание исходного `.java` файла
* упаковка результата в `.jar`
* работа через reflection API

---

## Интерфейсы

Основной интерфейс:

```java
public interface JarImpler extends Impler {
    void implementJar(Class<?> token, Path jarFile) throws ImplerException;
}
```

---

## Ограничения

* генерируются только публичные методы
* методы реализуются заглушками (default behavior)
* используется стандартная Java reflection
